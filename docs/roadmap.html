💼 Interview Prep Bank

CV-specific theory, implementation, and system-design questions with hints

Theory

Why does ResNet use skip connections?

Hint: Vanishing gradients + identity shortcuts let very deep networks train without degradation.

Implementation

Implement IoU from scratch in NumPy

Hint: intersection = max(0, x2−x1) × max(0, y2−y1). Union = A + B − intersection.

System Design

Design a real-time pedestrian detector for a city intersection camera

Hint: Consider latency SLA, edge vs. cloud, model selection, failure modes, and demographic bias.

Theory

How does attention differ from convolution?

Hint: Conv = local fixed receptive field. Attention = global, dynamic weight per token pair.

Debugging

Your detector has high precision but low recall. What's wrong?

Hint: Confidence threshold too high. Lower it and re-examine the precision-recall curve.

Implementation

Explain NMS and implement it in Python

Hint: Sort by confidence, suppress overlapping boxes (IoU > threshold), keep highest-confidence box.
